# budget

A simple budget app.
Available at: [corpas.nu/testings/budget/](https://corpas.nu/testings/budget/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

```
npm run lint
```
